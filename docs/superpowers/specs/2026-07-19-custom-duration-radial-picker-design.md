# 自定义时长:径向圆盘选择器

日期:2026-07-19 · 分支 `feat/duration-presets-ui`

## 目标

把选择页「自定义」按钮打开的时长面板,从「两个 number input(小时+分钟)」换成**安卓原生闹钟风格的径向圆盘选择器**:两段式手势选时长,移动端不弹软键盘、手感顺。

核心播放逻辑(淡变烤 WAV + 双 audio 轮换 + 三态机)**完全不动**,只替换「自定义时长」这一 UI 入口,最终仍产出 ms 交给现有 `selectDuration(ms)`。

## 交互范式(已与用户确认)

- **A · 径向圆盘**(对比过数字输入框 / iOS 滚轮 / 双滑块后选定)
- **小时圆盘:0–11 单环,0 在顶**(`0` 占 12 点钟位置,1–11 顺时针)。选 0 = 纯分钟定时器,无需特例。上限 11 小时(助眠足够,放弃原 0–23h 全范围)。
- **分钟圆盘:与小时盘同款视觉**——只摆 12 个数字 `0 / 5 / 10 / 15 / 20 / 25 / 30 / 35 / 40 / 45 / 50 / 55`,**无任何刻度线**,数字大小统一。
  - 拖动可达 **1 分钟精度**任意值(如 32 分);盘面只显示 5 分锚点,但**指针与顶部大数字反映真实精确值**。

## 面板布局(从上到下,**无标题**)

> 用户明确要求:去掉「自定义时长」标题。

1. **大数字 `H : M`**(如 `2 : 30`)——实时反映当前选择,与播放页倒计时同字体风格(细体、字距、紫色光晕)。
2. **「小时 / 分钟」切换标签**——两个胶囊,当前态高亮;可点回切修正。
3. **圆盘**(SVG)——12 个数字环绕,从圆心到所选值的指针 + 末端圆点。
4. **「开始」按钮**——全宽胶囊,主色渐变;`0 : 00` 时禁用。

## 两段式流程

1. 进面板:默认**小时态**,初值 `0 : 30`(沿用现有默认)。
2. 点/拖小时盘选 0–11 → **自动切到分钟态**。
3. 点/拖分钟盘选 0–59(拖动 1 分精度,数字锚点 5 分一档)。
4. 顶部「小时/分钟」标签可点回切修正任一段。
5. 点「开始」→ 计算 ms → `selectDuration(ms)` → 关闭面板开播。
6. 点遮罩空白处 / 返回 = **取消**,不开播。

## 规则与边界

- `H : 0 + M : 0`(`0 : 00`)→ 「开始」禁用。
- 小时粒度:**整数小时**(0–11),拖动吸附到最近 30°。
- 分钟粒度:**1 分钟**(0–59),拖动吸附到最近 6°;数字仅作 5 分锚点显示。
- 点选(tap)vs 拖动(drag):pointerdown 起算,移动超过阈值算拖动,否则当点选最近数字(分钟态点选则吸附到该 5 分锚点)。
- 触控/鼠标统一用 Pointer Events。
- 选中值接到 `selectDuration((H*60+M)*60000)`,与现有 `confirmCustom` 产出一致。

## 视觉风格(与现有面板一致)

- 沿用现有 overlay 毛玻璃遮罩、卡片 `rgba(30,22,56,.92)` + `border 1px rgba(255,255,255,.1)` + `border-radius 20px`。
- 圆盘配色:数字 `#cfc3f0`,选中数字 `#fff` 加粗,指针/圆点主色 `#a78bfa`,外圈细环 `rgba(255,255,255,.1)`。
- 响应式:圆盘尺寸 `clamp`,小屏可用(`max-width: 320px` 卡片不变)。
- 过渡:`fade` 进出 overlay 不变;圆盘指针移动用短过渡(`transition: transform .12s`)跟手。

## 不做(YAGNI)

- 不做 13–23 小时(放弃双环全天形态)。
- 不做「无限」档(无限仍由选择页预设/默认 idle 播放覆盖,自定义面板只产出有限时长)。
- 不做刻度线、不做 AM/PM、不做秒。
- 不动核心播放、不改进现有预设按钮。

## 实现要点(给实现者)

- 全部在 `src/App.vue` 单文件内:替换现有 `.overlay / .custom-card / .custom-inputs` 模板块与相关 style;`customH/customM/customValid/customOpen/confirmCustom` 复用,只改输入方式。
- 新增组件内状态:`stage`(`'h' | 'm'`),默认 `'h'`。
- 圆盘 SVG:半径、12 个数字坐标按角度预计算(0° 在顶,顺时针,每 30°);小时态数字 `[0..11]`,分钟态数字 `[0,5,10,...,55]`。
- 指针角度:`hour → h*30°`、`minute → m*6°`。
- 拖动选值:监听 SVG 容器 `pointerdown/move/up`(up 在 window),由指针相对圆心的 `atan2` 算角度 → `value = round(angle/step) % count`。
  - 小时 `step=30, count=12`;分钟 `step=6, count=60`(产出 0–59)。
- 选完小时(指针释放或点选)后 `stage='m'`。
- 大数字 `H : M` 用 `padStart(2,'0')` 格式化。
- 保留 `.toast`、Media Session、所有播放逻辑原样。

## 验收(用户主观上手)

- 圆盘点/拖跟手、吸附自然。
- 选 0 小时 + N 分钟能正常启动纯分钟定时。
- `0:00` 开始按钮禁用。
- 两段切换流畅、标签回切正常。
- 移动端不弹软键盘、熄屏播放不受影响(未触核心)。
